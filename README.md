# Animated Registration Form

A multi-step registration form with smooth animations and client-side validation, built with vanilla HTML, CSS, and JavaScript.

![Preview](preview.gif)

## Features

- **4-step form flow** — username, email, password, and a thank-you screen
- **Input validation** — username length check (≥6 chars), email regex validation
- **Visual feedback** — background color changes (green/red) and shake animation on invalid input
- **Smooth transitions** — CSS-powered slide-in/out animations between steps
- **No dependencies** — only Font Awesome 6 for icons (loaded via CDN)

## Tech Stack

| Technology | Role |
|---|---|
| HTML5 | Structure |
| CSS3 | Layout & animations |
| Vanilla JavaScript | Form logic & validation |
| Font Awesome 6 | Icons (CDN) |

## Getting Started

Since this is a static frontend project, no setup or build step is required:

1. Clone or download the repository
2. Open **`aze.html`** in any modern browser
3. Start typing in the username field and click the down-arrow to proceed
